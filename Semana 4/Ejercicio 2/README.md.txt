# Conversión de Números

## Descripción
Este programa convierte números enteros de archivos de texto a:
- Binario
- Hexadecimal

## Uso
Ejecutar en la terminal desde la carpeta donde se encuentran los archivos:
```sh
python convertNumbers.py TC1.txt TC2.txt TC3.txt TC4.txt
Salida
