# Conteo de Palabras

## Descripci√≥n
Este programa cuenta la cantidad de veces que aparece cada palabra en archivos de texto.

## Uso
Ejecutar en la terminal desde la carpeta donde se encuentran los archivos:

python wordCount.py TC1.txt TC2.txt TC3.txt TC4.txt TC5.txt


Salida
Se generan archivos con los resultados, como:

WordCountResults_TC1.txt
WordCountResults_TC2.txt
...
WordCountResults_TC5.txt
