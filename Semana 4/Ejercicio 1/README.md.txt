# Cálculo de Estadísticas

## Descripción
Este programa lee archivos de texto con números y calcula estadísticas:
- Media
- Mediana
- Moda
- Varianza
- Desviación estándar
- Tiempo de ejecución

## Uso
Ejecutar en la terminal desde la carpeta donde se encuentran los archivos:

python computeStatistics.py TC1.txt TC2.txt TC3.txt TC4.txt TC5.txt TC6.txt TC7.txt

Se generan archivos con los resultados, como:

StatisticsResults_TC1.txt
StatisticsResults_TC2.txt
...
StatisticsResults_TC7.txt
